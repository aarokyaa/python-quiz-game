# Python Quiz Application

A simple command-line quiz application built with Python. This project demonstrates basic programming concepts like classes, data handling, user input validation, and score tracking.

---

## Features
- Loads questions from a data source
- Presents questions interactively in the terminal
- Accepts user input (True/False)
- Validates inputs and provides instant feedback
- Keeps track of the user's score
- Displays final results at the end of the quiz

---

## How to Use
1. Download or clone the repository.
2. Ensure you have Python 3 installed on your system.
3. Run `main.py` in your terminal or IDE:
4. Answer questions by typing "True" or "False".
5. View your score at the end of the quiz.

---

## Files
- `main.py` : Main execution script
- `quiz_brain.py` : Handles quiz logic and flow
- `question_model.py` : Defines the Question class
- `data.py` : Contains the questions data in a list of dictionaries
